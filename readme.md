## Descripción

Aplicación educativa e interactiva sobre **Estadística y Probabilidad**, desarrollada a partir de los conocimientos adquiridos durante el cursado universitario de esta materia.

Lo realicé como una forma de **llevar la teoría a un proyecto concreto**, integrando explicaciones, navegación por contenidos, herramientas prácticas y una interfaz visual moderna.

Este proyecto tiene un valor académico y personal muy importante para mí, ya que fue desarrollado al cerrar la materia de **Estadística y Probabilidad con calificación 10**, transformando lo aprendido en una aplicación funcional, visual y presentable. **Cuenta con texto y símbolos flotantes**

## Objetivo del proyecto

El objetivo de este sistema es ofrecer una experiencia más dinámica, visual e interactiva para aprender y explorar contenidos fundamentales de Estadística y Probabilidad.

La idea fue unir:

- teoría
- secciones organizadas
- herramientas interactivas
- diseño moderno
- posibilidad de ejecución como aplicación web y también como programa de escritorio

---

## Funcionalidades principales

El sistema incluye distintas secciones y recursos, entre ellos:

- **Pantalla principal / inicio**
- **Sección de teoría y contenidos**
- **Conceptos básicos de Estadística y Probabilidad**
- **Teoremas y fórmulas importantes**
- **Variables estadísticas**
- **Calculadora estadística**
- **Simulador de probabilidad**
- **Sección de notas**
- **Navegación simple e intuitiva entre módulos**
- **Interfaz visual moderna y organizada**

---

## ¿Cómo está hecho el proyecto?

Este sistema combina Python con tecnologías web:

- **Flask** se utiliza para ejecutar y servir el proyecto localmente.
- **HTML, CSS y JavaScript** se utilizan para construir toda la interfaz visual.
- **Tailwind CSS** complementa el diseño y la maquetación.
- **Electron** permite que el proyecto también pueda adaptarse o ejecutarse como aplicación de escritorio.

Por eso, no es solamente una página HTML, sino una aplicación que puede presentarse como proyecto web y también como programa.

## Cómo ejecutar el proyecto

# Opción 1: Ejecutarlo con Flask

1. Clonar este repositorio y una vez ejecutado abrir en el navegador la IP tirada por la terminal del proyecto

---

# Opción 2: Ejecutarlo como aplicación de escritorio

El proyecto también incluye estructura relacionada con Electron, por lo que puede adaptarse o ejecutarse como programa de escritorio.

Para esa versión:

Instalar Node.js
Ejecutar:
npm install
npm start
